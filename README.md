# travelAgentChatbot
Travel Agent - Travel Information Chatbot

Travel Agent is a smart tourism information chatbot developed using Flask (Python), HTML, CSS, and JavaScript.
It provides instant travel-related information including visa requirements, vaccination details, currency information, emergency contacts, and local customs.
